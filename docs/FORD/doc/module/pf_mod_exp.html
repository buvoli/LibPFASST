Tommaso Buvoli this module extends pf_sweeper_t and is used for creating an exponential sweeper that solves equations of the form When extending this class, you must supply the functions phib, swpPhib, and resPhib that each compute matrix-vector products of the form in addition to the function f_eval for compluting the nonlinear term N(t,y). The complete description of these three functions is contained below.
