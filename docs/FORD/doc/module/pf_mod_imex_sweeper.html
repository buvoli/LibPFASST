Module for the IMEX Sweeper of the the derived sweeper class for doing IMEX sweeps for an equation of the form The piece is treated explicitly and implicitly Afer this sweeper is initialized (usually in main), the logical flags can be changed if desired

 explicit:  Make false if there is no explicit piece

 implicit:  Make false if there is no implicit piece

The user needs to supply the feval and fcomp routines for a given example
