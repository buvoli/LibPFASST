Module to define and encapsulation of a Petsc Vector.

When a new solution is created by a PFASST level, this encapsulation uses the levels 'shape_lev' attribute to create a new array with that shape. Thus, the 'shape'_lev attributes of the PFASST levels should be set appropriately.
