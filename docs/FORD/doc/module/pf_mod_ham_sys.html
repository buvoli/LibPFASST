Hamiltonian system encapsulation.

This encapsulation uses the levels 'lev_shape' attribute to create define 2 new arrays of size (ndim,nparticles) corresponding to the two components (p,q) in the Hamiltonian particles system

The helper routines get_array can be used to extract pointers to the different components of the system array performing any copies.
